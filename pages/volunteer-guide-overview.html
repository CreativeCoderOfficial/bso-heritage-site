📊 Google Sheets Columns Explained

The collection is driven by a Google Sheet with multiple tabs. Each tab serves a specific purpose, and they're all aggregated into a combined table that the website uses.

Remember: There are four main sections (Groups, Badges, Documents, Far & Wide), each with their own table in the Google Sheet. The combined table aggregates all of them for the website to use.

The essentials — every row needs these

objectid Unique ID for each item. Used internally to identify records, build links, and connect articles to metadata. Must be lowercase, no spaces, no slashes. Once set, never change it.
title Display name shown on browse cards, the map, search results, and the article page. Keep it concise and consistent.
format Tells the website what kind of file this is using MIME type codes like image/jpeg, application/pdf, audio/mpeg. Controls how the item is displayed (image viewer, PDF embed, audio player, etc.). See the MIME types reference table below for all supported formats.
filename Name of the actual file in the objects/ folder (e.g., grp-be-001.jpg). Leave blank for items with no file, or paste a full external URL if hosted elsewhere.

Classification — what kind of thing is it

type Broad category. Must be exactly one of: Group, Badge, Document, Far & Wide. Used by category pages and article navigation trail.
era Historical period the item belongs to (e.g., BGA, BSWE, BSO). Used for filtering and the subjects cloud.

Organisation and location — where does it belong

organisation Specific organisation name (e.g., 1st Brussels Group).
district BSO district (e.g., Brussels, Rhine).
country Country the item relates to (e.g., Belgium, Germany). Populates the country filter on browse page.
group_no Group's number (e.g., 1st, 3rd). Separate from organisation name for independent searching and filtering.
reg_no Regiment number for military-connected groups. Used for sorting and filtering regiment-specific items.

Group-specific detail

scarf_colour Colour of the group's neckscarf. Historical detail for groups.
date_founded When the group was established. Separate from general date field so date range filtering can check founding dates independently.
date_closed When the group closed, if applicable. Also checked by the date range filter.

Content and context

subject Topics or themes associated with the item, separated by semicolons (e.g., camping; jamboree; Baden-Powell). Powers the subjects cloud page and browse filtering.
latitude and longitude Geographic coordinates. Used to place the item as a pin on the map page. Leave blank if there's no meaningful location.
description Free-text summary of the item. Shown on the item page and indexed by search.

Badge-specific fields

badge_tier Level of the badge (e.g., Bronze, Silver, Gold, or programme stage like Scout, Explorer).
badge_type Category of badge (e.g., Activity, Staged, Challenge Award).
colour_variant For badges that come in multiple colour versions, note which variant this record represents.

Document and publication-specific fields

document_type Kind of document (e.g., Annual Report, Newsletter, Minutes, Programme).
date General date associated with the item (publication date, event date, etc.). Used by the timeline page. Format as YYYY, YYYY-MM, or YYYY-MM-DD.
issue_number For Far & Wide issues, the issue number.
season For Far & Wide issues, the season or term (e.g., Spring 1987, Autumn 2003).

Article linking

article_url Optional. If a volunteer has written a rich markdown article for this item, paste the permalink here (e.g., /articles/grp-be-001.html). When filled in, browse card clicks go directly to the article page instead of the standard item page.

📋 MIME Types Reference Table

Use these standard MIME type values in the format field to tell the website how to display your item. Look at which extension your file ends. Then copy the exact MIME type from this table.

Category MIME Type Description Example Files
Images image/jpeg JPEG image format .jpg, .jpeg
image/png PNG image format (supports transparency) .png
image/gif GIF image format (supports animation) .gif
image/svg+xml SVG vector image .svg
Documents application/pdf Adobe PDF document .pdf
text/plain Plain text document .txt
Audio audio/mpeg MP3 audio file .mp3
audio/wav WAV audio file (uncompressed) .wav
Video video/mp4 MP4 video file .mp4
video/webm WebM video format .webm
video/quicktime QuickTime/MOV video .mov
